# Extracción de Información de recibos y facturas
## *Descripción del Proyecto*

Este proyecto tiene como objetivo desarrollar un sistema capaz de extraer información estructurada a partir de recibos y facturas en formato físico (imágenes). Para ello, se emplean técnicas de procesamiento digital de imágenes y reconocimiento óptico de caracteres (OCR).

El sistema implementa etapas de preprocesamiento, detección de bordes, segmentación de regiones de interés y normalización de texto, con el fin de mejorar la calidad de la información antes de aplicar OCR. Finalmente, se busca identificar y extraer automáticamente datos relevantes como RUC, DNI, fecha e importe total.

Este repositorio corresponde al avance del 50% del proyecto, donde ya se cuenta con la implementación del procesamiento de imágenes y una primera integración funcional con OCR.


## *Estado del Proyecto*

Avance del 50%:

* Preprocesamiento de imágenes
* Detección de bordes
* Segmentación de texto en filas
* Normalización de regiones de interés
* Integración inicial con OCR
